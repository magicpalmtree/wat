## Array

Provides support for creation of arrays of any data type.

```js
var arr = [];
var arr = new Array();        
var arr = new Array([size]);                 // []
var arr = new Array([el0[, el1[, eln...]]]); // [1, 2, 3]
```

```js
var arr = ['a', 2, 'c'];
arr[1];                     // 2
```
